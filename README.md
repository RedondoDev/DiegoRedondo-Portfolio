# 🎞️ DiegoRedondo — Portfolio

## Creative Video Editor & Visual Storyteller

Personal portfolio website showcasing selected works by **Diego Redondo**, a freelance video editor specialized in advertising, music videos, and promotional audiovisual content.

The site is designed as a visually-driven, modern experience focused on motion, rhythm, and storytelling. It combines immersive video, smooth navigation, and a minimalist aesthetic to highlight each project clearly and effectively.
